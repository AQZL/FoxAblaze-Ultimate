# Fox Ablaze Ultimate

A NeoForge mod for Minecraft 1.21 that adds more Ultimate Skills to [Tensura](https://www.curseforge.com/minecraft/mc-mods/tensura).

## Features

- **Raphael (Wisdom King)** — Analysis, naming, and predation filter assistance
- **Beelzebub (Gluttony King)** — Spatial storage with fluid management, entity capture, and fusion ceremony
- **Uriel (Covenant King)** — Beam projectile and covenant abilities

## Dependencies

| Mod | Role |
|-----|------|
| [Tensura](https://www.curseforge.com/minecraft/mc-mods/tensura) | Required — base skill system |
| [ManasCore](https://www.curseforge.com/minecraft/mc-mods/manascore) | Required — skill API |
| [Architectury](https://www.curseforge.com/minecraft/mc-mods/architectury-api) | Required — networking |
| [EMI](https://www.curseforge.com/minecraft/mc-mods/emi) | Optional — UI compatibility |

## Building

```bash
./gradlew build
```

Output jar: `build/libs/foxablazeultimate-<version>.jar`
